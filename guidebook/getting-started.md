# Getting Started

## Getting The Initial Materials
To get started with Null, you will need a few basic materials. You will need a lot of iron, which you must smelt in a 

furnace to create <ItemLink id="steel_ingot" />. Want more yield from each smelting operation? Look at the [Dimensional Hot Plate](machines/Dimensional-Hot-Plate.md).
